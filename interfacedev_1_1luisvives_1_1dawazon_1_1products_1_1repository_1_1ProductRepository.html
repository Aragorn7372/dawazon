Descripción detallada

Repositorio para operaciones CRUD sobre la entidad Product.

Extiende JpaRepository para proporcionar métodos estándar de acceso a datos y define consultas personalizadas para operaciones específicas de productos.

Long  version

Resta stock de un producto de forma atómica.

Actualiza el stock solo si hay cantidad suficiente y el producto no está eliminado. Usa control de concurrencia optimista mediante version.

Parámetros
idID del producto.
amountCantidad a restar del stock.
versionVersión actual del producto para control de concurrencia.
Devuelve
Número de filas afectadas (1 si se actualizó, 0 si no).
